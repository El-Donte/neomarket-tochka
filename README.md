## Пересборка Docker
```
docker-compose down

docker-compose build --no-cache

docker-compose up
```

## Перед сборкой создайте .env файл
```
cp .env.example .env
```
Задайте в нем ваши параметры

## Инструкция для входа и создания:

### Залогиньтесь:

2. Используйте **ИНН:** `1122334455` и **пароль:** `testpass`.

### Создайте товар:
1. Перейдите в **Дашборд** (Dashboard).
2. Нажмите **+ Добавить товар**.
3. Заполните поля:
   - **Название товара:** `Cyber Glasses`
   - **Название вариации:** `Standard Black` (или любое другое)
   - **Цена:** `299`
   - **Описание:** `Premium smart glasses.`

### Проверьте:
1. Перейдите в **Каталог**. Теперь ваш товар отображается с ценой `$299`.
2. Нажмите **Добавить**, и вы увидите, что в корзине (иконка вверху) появилось число `1`.
